# 계량적 논리로 질문하고 데이터로 가치를 증명합니다

비즈니스 데이터 전략가 **이상민**입니다.  
국제통상 및 경영학적 전문 지식을 바탕으로 데이터 간의 인과관계를 규명하고, 기업의 내재가치를 정량화하는 분석에 집중하고 있습니다.

### 주요 프로젝트 및 성과

#### 대한항공 기업 가치평가 및 투자 전략 분석
- **설명:** 항공 산업 특수성을 반영한 DCF 및 APV 모델 기반 내재가치 산출
- **성과:** **WACC 3.89%** 및 시나리오별 적정 주가를 도출하여 논리적 투자 의견 제시

#### 야간 위성 조도 데이터를 활용한 국가별 GDP 예측
- **설명:** 계량경제학적 접근을 통해 비정형 데이터를 활용한 대안 경제 지표 설계
- **성과:** **$R^2=0.88$**의 높은 설명력을 확보하며 데이터의 비즈니스적 가치 확장 입증

#### 설명 가능한 AI(XAI) 기반 고객 이탈 예측
- **설명:** XGBoost 모델과 SHAP 분석을 결합하여 이탈의 핵심 동인 규명
- **성과:** 이탈률 **5.0%p** 저감 가이드라인 및 데이터 기반의 타겟 마케팅 전략 제안
